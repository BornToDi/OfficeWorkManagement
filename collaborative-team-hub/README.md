# Collaborative Team Hub

Monorepo scaffold for the Collaborative Team Hub technical assessment.

Structure:

- apps/web — Next.js frontend
- apps/api — Express backend
- packages/shared — shared constants and utilities

Run locally:

1. Install dependencies in repo root (use npm, pnpm or yarn)

```bash
npm install
```

2. Start dev: (requires installing per-app deps)

```bash
npm run dev
```
